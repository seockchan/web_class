ID 선택자

ID 선택기는 #점 문자가 아닌 문자로 시작하지만 클래스 선택기와 같은 방식으로 사용됩니다. 단, ID는 페이지당 한 번만 사용할 수 있으며 요소에는 하나의 id값만 적용할 수 있습니다. 세트 가 있는 요소를 선택할 수 있으며 id, ID 앞에 유형 선택기를 사용하여 요소와 ID가 모두 일치하는 경우에만 요소를 대상으로 지정할 수 있습니다.

Class 선택자

클래스 선택자는 점( .) 문자로 시작합니다. 해당 클래스가 적용된 문서의 모든 항목을 선택합니다.
