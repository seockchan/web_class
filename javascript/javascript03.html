02. 익명 함수

    const func = function(){
        document.write("함수가 실행되었습니다.");
    }
    func();
    }

결과보기

함수가 실행되었습니다.

    }

결과보기

04. 리턴값 함수

    {
    function func(){
        const str = "함수가 실행되었습니다."
        return str;
    }
    document.write(func());
    }

결과보기

함수가 실행되었습니다.
